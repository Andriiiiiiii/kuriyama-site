# Управление размерами шрифтов

Все размеры шрифтов на сайте управляются через единый конфигурационный файл: `src/config/typography.ts`

## Как изменить размеры шрифтов

Откройте файл `src/config/typography.ts` и измените значения:

```typescript
export const TYPOGRAPHY_CONFIG = {
  // Базовая ширина для нормализации (px)
  baseWidth: 1441.5,

  // Размеры шрифтов в пикселях (при baseWidth)
  fontSizes: {
    // Основной заголовок (80px при 1441.5px ширине)
    heading: 80,

    // Подзаголовок (30px при 1441.5px ширине)
    subheading: 30,

    // Обычный текст (22px при 1441.5px ширине)
    body: 22,
  },
} as const;
```

## Параметры конфигурации

### `baseWidth` (число)
Базовая ширина окна в пикселях, для которой заданы размеры шрифтов.
- **По умолчанию:** 1441.5
- **Рекомендуется:** не изменять, если не требуется другая базовая ширина

### `fontSizes.heading` (число)
Размер основных заголовков (h1, h2) в пикселях при базовой ширине.
- **По умолчанию:** 80
- **Используется для:** Заголовки "Посещение", "План Куриямы", "Что это за место?" и т.д.

### `fontSizes.subheading` (число)
Размер подзаголовков (h3, малых заголовков) в пикселях при базовой ширине.
- **По умолчанию:** 30
- **Используется для:** Названия активностей, разделов

### `fontSizes.body` (число)
Размер основного текста в пикселях при базовой ширине.
- **По умолчанию:** 22
- **Используется для:** Описания, параграфы, кнопки

## Примеры изменений

### Увеличить все шрифты на 20%

```typescript
fontSizes: {
  heading: 96,      // 80 * 1.2
  subheading: 36,   // 30 * 1.2
  body: 26,         // 22 * 1.2
}
```

### Уменьшить все шрифты на 10%

```typescript
fontSizes: {
  heading: 72,      // 80 * 0.9
  subheading: 27,   // 30 * 0.9
  body: 20,         // 22 * 0.9
}
```

### Увеличить только заголовки

```typescript
fontSizes: {
  heading: 100,     // было 80
  subheading: 30,   // без изменений
  body: 22,         // без изменений
}
```

## Как работает адаптивность

Шрифты автоматически масштабируются в зависимости от ширины окна по формуле:

```
fontSize = (заданный размер / baseWidth) * ширина окна
```

Например, при `heading: 80` и `baseWidth: 1441.5`:
- При ширине окна 1441.5px → шрифт 80px
- При ширине окна 720px → шрифт 40px
- При ширине окна 1920px → шрифт ≈106px

## Использование в компонентах

После изменения конфигурации все компоненты, использующие `FONT_SIZES`, автоматически обновятся:

```typescript
import { FONT_SIZES } from '@/config/typography';

<h1 style={{ fontSize: FONT_SIZES.heading }}>
  Заголовок
</h1>
```

## Важно

После изменения конфигурации:
1. Сохраните файл `src/config/typography.ts`
2. Приложение автоматически перезагрузится (hot reload)
3. Все шрифты обновятся на всех страницах
